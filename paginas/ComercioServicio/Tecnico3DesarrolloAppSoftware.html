Comercio y Servicio

Técnico 3 Desarrollo de Aplicaciones de Software

Descripción del programa

La persona estudiante en este programa adquiere competencias para desarrollar aplicaciones de software de diferentes tipos; por ejemplo: escritorio, web, mobile con acceso a base de datos para el cumplimiento de requerimientos establecidos por el cliente (con base en un análisis previo para establecer el diseño adecuado) con ética y profesionalismo. Cabe mencionar que, la persona interesada en este programa de estudio debe aprender a comunicarse con los clientes internos y externos de las empresas, para llegar a la necesidad del cliente y así diseñar aplicaciones de software que solucionen el problema o necesidad de este. Siendo fundamental durante su formación el trabajo en equipo dado a que en su trabajo tendrá que relacionarse con equipos interdisciplinarios (otros profesionales).

Requisitos de Ingreso

  • Edad: Mayor de 16 años.
  • Nivel académico: Egresado de quinto año.
  • Proceso de admisión: ( X ) Sí ( ) No

Habilidades de funcionalidad

Habilidades físicas

  • Trabajar durante muchas horas en el uso de la computadora. Motora fina para el uso de teclado y mouse.

Habilidades Cognitivas

  • Análisis de problemas, razonamiento lógico matemático, atención, orden, capacidad de memorizar términos y conceptos, concentración.

Habilidades sensoriales

  • Agudeza visual (monocular y binocular a corta y larga distancia)
  • Percepción de formas.
    • Habilidad visual, por ejemplo, para analizar líneas de código de programación y detectar símbolos o letras que estén generando un error en el programa.

Otras habilidades necesarias

  • Habilidades sociales son fundamentales: ser una persona con excelente actitud, responsabilidad, trabajar bien en equipo, tolerancia, respeto, buen vocabulario, buenos modales.

Tareas típicas que se realizan en el programa

  • Analizar problemas y plantear soluciones siguiendo una metodología, crear diagramas, elaborar informes técnicos.
  • Utilizar software de edición multimedia (imágenes de mapas de bits, imágenes vectoriales).
  • Utilizar software de creación de prototipos (mockups, wireframes).
  • Programar en HTML, CSS, javascript.
  • Trabajar con metodologías de administración de proyectos.

Plan de estudio

  • Producción de componentes multimedia.
  • Diseño de interfaces web.
  • Codificación de páginas web.
  • Diseño de algoritmos.
  • Programación web en el entorno cliente.
  • Uso de sistemas de gestión de contenidos.
  • Publicación de páginas web.
  • Planificación de proyectos web.
  • Desarrollo de proyectos web en el entorno.

Perfil de salida

El egresado será capaz de crear componentes multimedia, diseñar interfaces web atractivas, y plantear algoritmos en pseudocódigo, diagramas y código. También dominará la programación en HTML, CSS, y JavaScript, utilizará sistemas de gestión de contenidos, publicará páginas web, y planificará proyectos web de manera efectiva.

Mercado laboral

  • Empresas privadas o instituciones públicas que cuenten con un área de desarrollo Web.
